# Retail Sales Analysis (SQL Project)

## **Project Overview**

This project demonstrates SQL skills by analyzing retail sales data. It includes database creation, data cleaning, and solving business problems using SQL queries.

## **Tools Used**

* MySQL
* SQL

## **Database Details**

* Database Name: `sql_project_p2`
* Table Name: `retail_sales`

## **Data Cleaning**

* Checked for NULL values
* Removed incomplete records

## **Key Analysis**

* Total sales and transactions
* Unique customers
* Category-wise performance
* High-value transactions
* Customer insights

## **Business Questions Solved**

* Sales on specific dates
* Category-based filtering
* Total sales by category
* Average customer age
* Top 5 customers
* Monthly sales trends
* Shift-based sales analysis

## **How to Run**

1. Create database:

   ```sql
   CREATE DATABASE sql_project_p2;
   ```

2. Create table and import dataset (CSV)

3. Run SQL queries
